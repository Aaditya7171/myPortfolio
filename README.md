# Aditya Tomar Portfolio

This is the personal portfolio website for Aditya Tomar, showcasing skills, projects, and contact information.

## Features

- Responsive design
- Dark/light theme toggle
- Project showcase
- Skills section
- Photography gallery
- Contact form
- Resume download

## Technologies Used

- React
- TypeScript
- Tailwind CSS
- Framer Motion
- Vite

## Deployment

This website is deployed on GitHub Pages.
